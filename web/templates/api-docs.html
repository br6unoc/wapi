API

Integre o BotWapp com seus sistemas externos

Autenticação

Os endpoints de envio usam API Key no header apikey.

Cada conexão tem sua própria API Key, disponível na aba Testar ou via GET /instances/:nome.

URL base:

POST /instances/:nome/send/text

Envia uma mensagem de texto para um número WhatsApp.

Headers

HeaderValorObrigatório
apikeyAPI Key da conexãoSim
Content-Typeapplication/jsonSim

Body (JSON)

CampoTipoDescrição
numberstringNúmero com código do país, sem +. Ex: 5511999999999
messagestringTexto da mensagem

Exemplo

curl -X POST /instances/minha-conexao/send/text \
  -H "apikey: SUA-API-KEY" \
  -H "Content-Type: application/json" \
  -d '{"number":"5511999999999","message":"Olá! Como posso ajudar?"}'
const response = await fetch('/instances/minha-conexao/send/text', {
  method: 'POST',
  headers: {
    'apikey': 'SUA-API-KEY',
    'Content-Type': 'application/json',
  },
  body: JSON.stringify({
    number: '5511999999999',
    message: 'Olá! Como posso ajudar?',
  }),
});

const data = await response.json();
console.log(data);
import requests

url = "/instances/minha-conexao/send/text"
headers = {
    "apikey": "SUA-API-KEY",
    "Content-Type": "application/json",
}
payload = {
    "number": "5511999999999",
    "message": "Olá! Como posso ajudar?",
}

response = requests.post(url, json=payload, headers=headers)
print(response.json())

Resposta (200)

{"ok": true, "id": "uuid-da-mensagem"}
POST /instances/:nome/send/media

Envia imagem, vídeo, áudio ou documento. Usa multipart/form-data.

Campos (form-data)

CampoTipoDescrição
numberstringNúmero com DDI
filefileArquivo (imagem, vídeo, áudio, documento)
captionstringLegenda (opcional)

Exemplo

curl -X POST /instances/minha-conexao/send/media \
  -H "apikey: SUA-API-KEY" \
  -F "number=5511999999999" \
  -F "file=@/caminho/imagem.jpg" \
  -F "caption=Confira nossa oferta!"
import requests

url = "/instances/minha-conexao/send/media"
files = {"file": open("imagem.jpg", "rb")}
data  = {"number": "5511999999999", "caption": "Confira nossa oferta!"}
headers = {"apikey": "SUA-API-KEY"}

r = requests.post(url, files=files, data=data, headers=headers)
print(r.json())
GET /instances/:nome/status

Verifica se a conexão está ativa. Não requer autenticação.

curl /instances/minha-conexao/status

Resposta (200)

{"status": "connected", "phone": "5511999999999"}

Envia mídia a partir de uma URL pública.
